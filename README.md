# Geometry 

This is a library of geometric calculations for neuroimaging. For now, just circles.  

## Contributing

We welcome contributions from the community. Please create a fork of the project on GitHub and use a pull request to propose your changes.

Before proposing a change through a pull request, please post an issue to Github.

## Getting help

Please post issues on the project GitHub page. Or don't, up to you. We don't care what you do and we aren't going to maintain this repo.
